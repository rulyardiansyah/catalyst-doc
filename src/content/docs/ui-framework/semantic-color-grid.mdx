Serupa dengan [Semantic Typography](./semantic-typography), token-token berikut harus ditampung di dalam direktori project karena semua komponen dan elemen visual lainnya mengacu pada table ini sebagai referensinya.

## Text Color

**Prefix**: `alias.text-color.`

| Nama Token            | Deskripsi                                                                                                                                |
| --------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| `default`             | Warna default untuk semua elemen teks, kecuali yang didefinisikan secara khusus.<br/>`{palette.light.neutral.neutral.cement.500}`        |
| `subtle`              | `{palette.light.neutral.slate.300}`                                                                                                      |
| `light`               | Warna untuk semua elemen teks yang akan tetap terang walaupun berubah tema dark/light.<br/>`{palette.light.neutral.neutral.coral-white}` |
| `dark`                | Warna untuk semua elemen teks yang akan tetap gelap walaupun berubah tema dark/light.<br/>`{palette.light.neutral.neutral.dark-slate}`   |
| `brand-default-begin` | `{palette.light.primary.cyan.500}`                                                                                                       |
| `brand-default-end`   | `{palette.light.primary.sky-blue.500}`                                                                                                   |
| `brand-hover-begin`   | `{palette.light.primary.cyan.500}`                                                                                                       |
| `brand-hover-end`     | `{palette.light.primary.sky-blue.500}`                                                                                                   |
| `interactive`         | `{palette.light.primary.cyan.600}`                                                                                                       |
| `informative`         | `{palette.light.primary.sky-blue.600}`                                                                                                   |
| `success`             | `{palette.light.complementary.grass-green.500}`                                                                                          |
| `warning`             | `{palette.light.complementary.sunset.500}`                                                                                               |
| `destructive`         | `{palette.light.complementary.paradise-pink.500}`                                                                                        |
| `highlight`           | `{palette.light.neutral.neutral.coral-white}`                                                                                            |
