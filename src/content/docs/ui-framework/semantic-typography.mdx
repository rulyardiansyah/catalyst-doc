---
title: Semantic Typography
---

Token-token berikut dibuat standard (mandatori) di semua project, sehingga implementasinya harus ditampung dalam suatu variabel atau sejenisnya.

**Prefix**: `alias.text-group.`

| Nama Token          | Deskripsi                                                                                                                                    |
| ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| `body`              | Digunakan untuk paragraf, caption, list, dsb..<br/>Default value: `typeface.maven-pro`                                                       |
| `code`              | Digunakan untuk semua yang berhubungan dengan kode.<br/>Default value: `typeface.ubuntu-mono`                                                |
| `header-display`    | Digunakan untuk elemen teks dengan ukuran sangat besar, seperti Display dan Hero.<br/>Default value: `typeface.bignoodletitling`             |
| `header-heading`    | Digunakan untuk elemen teks heading, seperti Heading 1, Heading 2, dst..<br/>Default value: `{alias.text-group.body}`                        |
| `numeric-repeat`    | Digunakan untuk elemen teks yang menunjukkan nominal dan digunakan di banyak tempat atau berulang.<br/>Default value: `typeface.ubuntu-mono` |
| `numeric-highlight` | Digunakan untuk elemen teks yang menunjukkan angka atau nominal dan digunakan sedikit tempat.<br/>Default value: `typeface.bignoodletitling` |
| `quotation`         | Digunakan untuk elemen teks kutipan.<br/>Default value: `typeface.merriweather`                                                              |
